## Batch Download Strategies

### Option 1: Enable Sci-Hub in Download Script

```python
# Edit paper_download_script.py line ~380
USE_SCIHUB = True  # Enable Sci-Hub fallback
```

Then re-run: `python paper_download_script.py`

### Option 2: University Library Access

If you have institutional access:
1. Connect to university VPN
2. Visit library website
3. Use DOI links provided above
4. Download PDFs manually

### Option 3: Request from Authors

Many authors provide PDFs upon request:
1. Find paper on Google Scholar
2. Click author profile
3. Send polite email requesting PDF
4. Most respond within 24-48 hours

### Option 4: Inter-Library Loan (ILL)

Free service at most universities:
1. Request through library website
2. Provide DOI/PMID
3. Usually receive within 1-3 days
